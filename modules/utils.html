modules.utils.get_logging_config() → None¶

Get the logging config options from config.ini, and apply them

Look for artifacts/config.ini and check the logging section for config options. If they are found, apply them. If interpolation is required, interpolate. For example, if the line:

log_path=$LOGS/$DATETIME.log

is found, replace $LOGS and $DATETIME with the absolute path to artifacts/logs and the current datetime, respectively. The datetime format used is %Y_%m_%d_%H_%M_%S
